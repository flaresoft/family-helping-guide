# 아버지 요양원 입소 안내 (family-helping-guide)

73세 어르신도 원격에서 언제든 접속해 큰 글씨로 보고, 직접 눌러가며 이해할 수 있도록 만든 **반응형 인터랙티브 안내 페이지**입니다. 요양원 입소 검증 보고서의 5가지 핵심 내용을 쉽게 정리했습니다.

## 무엇이 들어있나요

- **5가지 사실 확인** — 제목을 누르면 펼쳐지는 카드 (사실 / 조건부 / 잘못 배지)
- **본인부담금 계산기** — 수급자 유형·등급 유무를 누르면 부담 비율이 바로 바뀜
- **전입신고 시뮬레이터** — 요양시설 vs 가족 집 전입 결과 비교
- **타 지역 입소 절차** — 강릉 → 안양 단계별 흐름도
- **○✗ 오해 바로잡기** — 자주 헷갈리는 4문제
- **우리가 할 일 체크리스트** — 체크 상태가 기기에 저장됨

## 어르신 친화 설계

- 큰 기본 글씨(22px)와 **글자 크기 조절 버튼**
- 높은 대비 색상, 큰 터치 영역, 마우스 오버 없이 동작
- 휴대폰·태블릿·PC 어디서나 보이는 반응형
- 외부 라이브러리 없는 단일 HTML 파일 (오프라인에서도 열림)

## 보는 방법

1. **바로 열기**: `index.html` 파일을 브라우저로 열면 됩니다.
2. **원격 접속(권장)**: GitHub Pages로 게시하면 어디서든 링크로 접속 가능합니다.
   - 저장소 **Settings → Pages → Branch**를 `claude/file-analysis-tt7xgt`(또는 병합 후 `main`) `/ (root)`로 설정
   - 잠시 후 발급되는 주소를 어르신께 전달
